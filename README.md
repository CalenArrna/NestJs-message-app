## Description

This project is a simple api, that has 3 endpoints with simple messages, created by NestJs for tutorial purpose. 
The 3 endpoints are create, getById and getAll. The api store it in a local json file.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
